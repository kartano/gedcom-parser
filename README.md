# Gedcom Parser
GEDCOM parser library designed around the Gedcom 5.5 standard.

## Installation

Install the latest version with

```bash
$ composer require kartano/gedcom-parser
```

## About

### Requirements

- Gedcom-parser works with PHP 7.2 or above.
